# challenge-amigo_secreto_romi_alcoba
# Análisis de Tiendas - Informe para el Sr. Juan

Este proyecto contiene un análisis exhaustivo de cuatro tiendas con el objetivo de recomendar al Sr. Juan cuál de ellas debería vender. Se han evaluado diversos factores clave para cada tienda, incluyendo ingresos totales, ventas por categoría, calificaciones promedio de los clientes, productos más y menos vendidos, y costos de envío promedio.

## Estructura del Proyecto

El notebook está organizado en las siguientes secciones:

1.  **Importación de datos:** Carga de los archivos CSV de cada tienda en DataFrames de Pandas.
2.  **Análisis de facturación:** Cálculo y visualización del ingreso total por tienda.
3.  **Ventas por categoría:** Identificación y visualización de las ventas de productos por categoría en cada tienda.
4.  **Calificación promedio de la tienda:** Cálculo y visualización de la satisfacción del cliente a través de calificaciones promedio.
5.  **Productos más y menos vendidos:** Determinación de los productos con mayor y menor volumen de ventas en cada establecimiento.
6.  **Envío promedio por tienda:** Cálculo y visualización del costo de envío promedio por tienda.
7.  **Visualización:** Creación de gráficos utilizando `matplotlib` para representar los hallazgos clave de manera clara.
8.  **Informe final:** Una sección de texto que sintetiza todos los análisis y presenta una recomendación justificada al Sr. Juan.

## Cómo Utilizar el Notebook

Para ejecutar este análisis, sigue estos pasos:

1.  Abre el archivo `.ipynb` en Google Colab o cualquier entorno compatible con Jupyter.
2.  Asegúrate de tener instaladas las bibliotecas `pandas` y `matplotlib`. Si no, puedes instalarlas usando:
    ```bash
    pip install pandas matplotlib
    ```
3.  Ejecuta todas las celdas del notebook en orden. Esto importará los datos, realizará los cálculos, generará las visualizaciones y mostrará el informe final.

## Hallazgos Clave

El informe final contiene una recomendación detallada para el Sr. Juan, basada en el equilibrio entre el rendimiento financiero, la satisfacción del cliente y la eficiencia operativa de cada tienda.
